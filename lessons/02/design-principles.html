Visual Hierarchy

Facebook by Meta

facebook.com

Facebook is an amazing example of Visual Hierarchy in web design. The picture above is divided into 3 secitons, the page header, the user and friend stories and the posts. The purpose of facebook audience is to check the posts from their friends, pages they follow, etc. The amount of space taken by each section clearly comply with the purpose of the page audience, wich is to check the posts.

Repetition

Twitter Inc.

twitter.com

Twitter is a good example of repetition design principle. Each post repeat the same design so it is easy for users identify the begining of a new post and the end of the same. If you see the picture above, the different sections are repeated acros the twits. This example help the people quickly identify where to look for specific info. If they want to see who created the post, the description and post time, they go to the red section, the blue is the content section and the green is the user interaction section.

Proximity

Instagram by Meta

instagram.com

All the websites we have seen use some standard design principles. If we take a closer look to each one of them they all use the principle of proximity. An example of proximity can be seen in instagram as well. If you check the marks between each element you will notice that there is some standard space between each element, example is the red mark, this defined a space between posts, the green are the spaces between text inside a post and the blue are separation of specific sections between the post.
